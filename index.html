What we do

We generate and publish kid-safe short videos for creators and educational publishers.

How it works

  1. Generate a kid-safe script and storyboard.
  2. Create a vertical short video (9:16, ~20–25 seconds).
  3. Prepare captions and hashtags.
  4. User connects TikTok via OAuth consent.
  5. Publish via TikTok Content Posting API and track status.

TikTok integration

  • Login (OAuth): user authorizes account connection.
  • Scope: video.publish (only what we need for posting).
  • Posting: Direct Post init → upload/URL → status fetch until complete.

Kids-safety

  • No adult content, violence, hate, or harassment.
  • Positive themes: friendship, learning, humor, curiosity.
  • Content is generated with strict guidelines before publishing.
